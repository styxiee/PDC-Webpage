Week 13: Cloud Computing

Theoretical Foundations

Cloud computing relies on resource sharing and elasticity, enabling users to access scalable resources on demand.

Cloud Computing Models

  • IaaS: Provides virtualized computing resources over the internet.
  • PaaS: Offers a platform for building, testing, and deploying applications.
  • SaaS: Delivers software applications over the internet on a subscription basis.

Virtualization and Containerization

Virtualization creates virtual machines, while containerization packages applications and their dependencies into lightweight containers.

Week 14: Edge Computing and IoT

Edge Computing

Edge computing processes data closer to the source, reducing latency and improving performance.

Comparison with Cloud Computing

Unlike cloud computing, which relies on centralized data centers, edge computing operates at or near data sources.

IoT Environments

Distributed computing in IoT involves managing data from numerous interconnected devices, enabling smart systems like cities and autonomous vehicles.
